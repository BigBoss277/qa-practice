# Checklist

## Input validation
- Valid input
- Empty input
- Long input
- Leading/trailing spaces
- Special characters
- Mixed language input
- Numbers-only input
- Copy/paste input
- Uppercase/lowercase input
- Very short query
- Query with quotation marks
- Gibberish / meaningless input

## Search submission
- Repeated click on search button
- Pressing Enter instead of clicking button

## Navigation and results behavior
- Back behavior after opening results
- Refresh behavior on results page
- Search suggestions / autocomplete behavior
- Error message clarity if applicable
- Input field remains usable after invalid or unusual input
- Mobile/desktop difference if relevant
