# Shopping Cart Testing Project

## Test Object

https://sauce-demo.myshopify.com

## Feature Tested

Product catalog, product details, search, and shopping cart flow.

## Role

Manual QA tester.

## Environment

- Opera GX 130.0.5847.58
- Windows Desktop

## Testing Scope

- Opening home page and catalog page
- Viewing product details
- Adding products to cart
- Verifying product name, price, and quantity in cart
- Removing products from cart
- Checking cart behavior after page refresh
- Searching for existing and non-existing products
- Checking quantity validation behavior

## Artifacts

- `test-plan-lite.md` - testing goal, scope, risks, and criteria
- `checklist.md` - high-level coverage checklist
- `test-cases.md` - detailed test cases with actual results and statuses
- `issues.md` - documented bug report / validation issue
- `exploratory-notes.md` - exploratory testing notes and observations

## Summary

The main shopping cart flow worked as expected during testing. Product navigation, adding products to cart, removing products, and search were stable. One validation issue was found when entering a very large quantity value in the cart.
