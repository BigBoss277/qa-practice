# Test Plan Lite — Project 03: Shopping Cart

## Project
Third QA practice project

## Feature
Shopping Cart / Product Add-to-Cart Flow

## Test Object
https://sauce-demo.myshopify.com

## Goal of Testing
Check basic shopping cart functionality and identify possible functional, validation, UX, and exploratory issues.

## Scope
Testing will cover:
- opening the home page and catalog page
- viewing product cards
- opening product details
- adding products to cart
- checking cart page
- changing product quantity if available
- removing products from cart if available
- checking cart total / subtotal
- using search for product-related queries
- basic navigation between home, catalog, product page, and cart

## Out of Scope
- real payment testing
- completing a real order
- backend logic and database validation
- security testing
- performance testing
- full cross-browser testing beyond the selected environment

## Test Types
- Functional testing
- Validation testing
- UX testing
- Exploratory testing

## Environment
Opera GX 130.0.5847.58, Windows

## Risks
- This is a demo practice website, so some behaviors may be intentionally simplified
- Some features may not behave like a full production e-commerce website
- Checkout/payment flow should not be completed as a real purchase

## Entry Criteria
- Website is accessible
- Browser and internet connection are working
- Product pages are visible
- Cart is accessible
- Add-to-cart functionality is available

## Exit Criteria
- Core checklist items are completed
- Main test cases are executed
- Observed issues and UX observations are documented
- No real purchase/payment is completed
